Зайти на dd

Зеркала

Официальный список зеркал Диди всегда опубликован на самом dd Marketplace и подписан PGP ключом сайта. (смотри раздел "Помощь")

dd Marketplace имеет только один адрес в сети Тор и не использует .onion зеркал.

Лучший способ получить публичный PGP ключ Диди - это скачать его прямо с сайта dd Marketplace (из раздела "Помощь") через Tor Browser. Для работы с PGP потребуется компьютер.

На данный момент у Диди также есть официальный Tor2Web шлюз по адресу ddru.cc, работающий с любого устройства и через любой браузер:

Перейти на ddru.cc
